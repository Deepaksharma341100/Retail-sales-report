# Retail Sales Data Analysis Project

## Project Overview
This project analyzes retail sales data to understand customer behavior, product performance, and sales trends.  
The analysis is performed using **SQL** for querying the dataset and **Power BI** for interactive dashboards.  
The goal is to provide actionable insights and business recommendations based on real sales data.

---

## Dataset
- **File Name:** `retail_sales_dataset.csv`  
- **Number of Records:** 1000  
- **Key Columns:**
  - `Transaction ID`
  - `Date`
  - `Product Category`
  - `Gender`
  - `Age`
  - `Quantity`
  - `Total Amount`

---

## Tools & Technologies
- **MySQL:** For data cleaning, aggregation, and analysis
- **Power BI:** For interactive visualization and dashboard creation
- **Microsoft Word:** For project documentation

---

## SQL Analysis Performed
- **Total Sales & Orders**
- **Category-wise Sales & Average Order Value (AOV)**
- **Gender-wise and Age-wise Sales Analysis**
- **Age group × Product Category Analysis**
- **Time-based Analysis** (Daily & Monthly trends)
- **Key metrics calculated:** Total Sales, Total Orders, Average Order Value

## Key Insights
- Customers aged **Above 50** generate the highest revenue.
- **Beauty products** have the highest Average Order Value (AOV), indicating premium purchases.
- **Electronics** category generates the highest total sales, while **Clothing** leads in total order count.
- Sales peak during **May, October, and December**, reflecting seasonal and festival shopping trends.

---

## Business Recommendations
- Introduce **premium product offerings** and exclusive deals for customers above 50.
- Promote **beauty bundles and high-end products** to maximize revenue from premium customers.
- Plan **marketing campaigns and inventory** around festival months (October, December, May).
- Offer **discounts and loyalty programs** for 20–30 age group to increase purchase frequency.
